# FakeAgent Gradio 界面

这是一个基于 Gradio 的 Web 界面，用于展示 FakeAgent 虚假视频检测系统的分析结果。

## 功能特点

- 简洁直观的用户界面
- 支持视频上传和分析
- 分类展示分析结果
- 支持结果导出为 JSON 格式
- 提供示例视频快速测试

## 使用方法

### 1. 安装依赖

```bash
pip install gradio
```

### 2. 启动界面

```bash
# 使用默认设置启动
python run_gradio.py

# 或者直接运行
python gradio_app.py

# 自定义端口和主机
python run_gradio.py --port 8080 --host 127.0.0.1

# 创建公共分享链接
python run_gradio.py --share

# 启用调试模式
python run_gradio.py --debug
```

### 3. 使用界面

1. 在浏览器中打开显示的地址（默认为 http://0.0.0.0:7860）
2. 上传视频文件
3. 可选：输入视频标题
4. 选择是否启用预处理（推荐启用）
5. 点击"开始分析"按钮
6. 等待分析完成，查看各个标签页中的结果
7. 可点击"导出结果"下载完整的分析结果

## 界面说明

### 输入区域
- **视频上传**：支持常见视频格式
- **视频标题**：可选，有助于提高分析准确性
- **预处理选项**：推荐启用，可提高分析质量

### 结果展示

界面分为四个标签页，展示不同类型的分析结果：

1. **转录与帧描述**
   - 音频转录：视频中的语音内容
   - 帧描述：视频关键帧的视觉描述

2. **分析结果**
   - 一致性分析：音频与视觉内容的一致性
   - AI检测：检测是否由AI生成
   - 冒犯性语言检测：检测不当内容

3. **事实核查**
   - 事实核查：验证内容真实性
   - 外部证据：相关的外部信息
   - 可疑片段定位：标记可疑时间点

4. **最终分析**
   - 综合分析结果：所有分析的综合结论

## 注意事项

- 视频分析可能需要较长时间，请耐心等待
- 大型视频文件可能需要更多处理时间
- 分析结果仅供参考，不构成绝对判断
- 请确保上传的视频文件符合相关法律法规

## 故障排除

如果遇到问题，请检查：

1. 是否安装了所有必要的依赖
2. 视频文件格式是否受支持
3. 系统是否有足够的内存和存储空间
4. 网络连接是否正常（如果使用外部证据检索）
